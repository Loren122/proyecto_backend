Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar una aplicación de gestión de biblioteca utilizando tecnologías backend modernas.

En este proyecto aprendimos a integrar Django para la gestión de datos y API RESTful para la comunicación entre frontend y backend.
